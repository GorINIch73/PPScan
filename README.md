# PPScan

Android-приложение для сканирования и распознавания платёжных поручений с помощью OCR (Tesseract.js).

## Возможности

- Сканирование платёжных документов через камеру или галерею
- Автоматическое распознавание реквизитов:
  - Номер и дата поручения
  - Сумма платежа
  - Данные плательщика и получателя (наименование, ИНН, КПП, расчётный счёт, БИК, банк)
  - Назначение платежа
- Сохранение сканов в локальной базе данных
- Ручная проверка и редактирование распознанных данных
- Экспорт в CSV
- Просмотр RAW-текста OCR для отладки

## Стек

- **Frontend**: Vue 3 + Quasar Framework + TypeScript
- **Mobile**: Capacitor (Android)
- **OCR**: Tesseract.js (русский язык)
- **База данных**: Dexie.js (IndexedDB)
- **Сборка**: Vite

## Установка и запуск

```bash
# Установка зависимостей
cd ppscan-app
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для Android
npm run build
npx cap sync android
```

## Сканирование реквизитов

Приложение оптимизировано для документов из СберБизнес и распознаёт:
- Платёжные поручения формата СберБизнес
- ФИО в формате "Индивидуальный Предприниматель Фамилия Имя Отчество"
- Назначение платежа с НДС и без
