# PPScan

Android-приложение для сканирования и распознавания платёжных поручений с помощью OCR (Tesseract.js).

## Возможности

- Сканирование платёжных документов через камеру или галерею
- Автоматическое распознавание реквизитов:
  - Номер и дата поручения
  - Сумма платежа (цифрами и прописью)
  - Данные плательщика и получателя (наименование, ИНН, КПП, расчётный счёт, БИК, банк)
  - Назначение платежа
- Предобработка изображения (повышение контраста)
- Сохранение сканов в локальной базе данных
- Ручная проверка и редактирование распознанных данных
- Экспорт в CSV (сохранение в папку Documents)
- Просмотр RAW-текста OCR для отладки

## Стек

- **Frontend**: Vue 3 + Quasar Framework + TypeScript
- **Mobile**: Capacitor (Android)
- **OCR**: Tesseract.js (русский язык)
- **База данных**: Dexie.js (IndexedDB)
- **Сборка**: Vite

## Установка и запуск

```bash
# Установка зависимостей
cd ppscan-app
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для Android
./build.sh   # Linux/Mac
# или
build.bat    # Windows
```

## Сборка APK

После сборки откройте папку `android/` в Android Studio и соберите APK.

## Поддерживаемые форматы

Приложение оптимизировано для российских банковских документов:
- СберБизнес (основной формат)
- Документы с меткой "Назначение платежа" в конце секции
- ФИО в формате "Индивидуальный Предприниматель Фамилия Имя Отчество"
- Организации: ООО, АО, ЗАО, ИП

## OCR

- Предобработка изображения через Canvas (повышение контраста)
- Приоритетная система извлечения назначения платежа
- Поддержка OCR-артефактов (искажённые символы)
