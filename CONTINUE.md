# PPScan - продолжение работы

## Что сделано (2026-03-29)

### OCR
- Приоритетная система извлечения назначения платежа (4 метода)
- Поддержка формата СберБизнес ("Назначение платежа" - метка конца секции)
- Поддержка OCR-артефактов ([Бик [, | вместо И)
- Извлечение ФИО для ИП ("Индивидуальный Предприниматель")

### Предобработка изображения
- Canvas-based контрастное усиление (без OpenCV)
- Бок о бок превью оригинал/обработанное
- Обработанное изображение показывается по умолчанию

### Экспорт CSV
- Сохранение в папку Documents (Android)
- Имя файла с временной меткой: `payments_2026-03-29T15-30-45.csv`
- Очистка спецсимволов (переносы, табуляция, возврат каретки)
- Все поля в кавычках, экранирование кавычек

### Скрипты сборки
- `build.bat` - Windows
- `build.sh` - Linux/Mac

## Файлы

- `ppscan-app/src/services/ocr.ts` - OCR логика (~640 строк)
- `ppscan-app/src/services/imageProcessor.ts` - предобработка изображения
- `ppscan-app/src/services/export.ts` - экспорт CSV
- `ppscan-app/src/pages/ScanPage.vue` - страница сканирования
- `ppscan-app/src/types/index.ts` - типы PaymentFields, PaymentOrder
