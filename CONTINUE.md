# PPScan - продолжение работы

## Что сделано
- Создан Android-клон Приват24 для сканирования банковских платёжных поручений
- Приложение: Capacitor + Vue + Quasar
- OCR через Tesseract.js (русский язык)
- Хранение в IndexedDB (Dexie.js)
- Экспорт в CSV

### Проблемы для решения
1. **payerBik** - пусто, должен быть "047003608"
   - Паттерн: `/(?:бик)[^\d\w]*(\d{9})/gi` не находит из-за `[Бик [`
   - Нужен паттерн: `/бик[^\w]*[\[]?(\d{9})/gi`

2. **recipient** - "Индивидуальный Предприниматель Корнеев"
   - Лишние слова, должно быть "ИП Корнеев Сергей Анатольевич"
   - Нужно изменить регулярку для извлечения ФИО

3. **paymentPurpose** - пусто
   - Текст "Оплата за услуги..." находится перед "Назначение платежа"
   - Нужен поиск строки с "оплата" перед "Назначение платежа"
